# Анализ оттока клиентов телеком‑оператора (Power BI)

## Цель
Определить уровень оттока (churn) и выявить факторы, связанные с уходом клиентов, на основе открытого датасета Telco Customer Churn.

## Данные
- Источник: Kaggle — Telco Customer Churn
- Объем: 7 043 клиента, 21 признак (тип контракта, услуги, платежи, отток и др.)

## Инструменты
- Power BI Desktop
- Основные визуализации: круговая диаграмма, столбчатые диаграммы, матрица, слайсер

## Ключевые выводы
1. Общий уровень оттока составляет ~27% клиентской базы (1869 из 7043 клиентов).
2. Существенной разницы в оттоке по полу не наблюдается — мужчины и женщины уходят сопоставимо.
3. Наиболее лояльные клиенты — с двухлетним контрактом (Two year): они обслуживаются дольше всех и крайне редко уходят (отток ~3%).
4. Основной вклад в отток дают клиенты с помесячным контрактом (Month‑to‑month): их средний срок жизни минимален, а доля оттока максимальна (около 43%).
5. Для снижения оттока целесообразно стимулировать переход с помесячных контрактов на годовые/двухлетние и предлагать удерживающие акции группе высокого риска.
