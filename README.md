# **Анализ комплексного набора данных о дорожно-транспортных происшествиях**
## 1. Описание Проекта
Данный проект представляет собой всесторонний анализ данных о дорожно-транспортных происшествиях, целью которого является выявление ключевых факторов, влияющих на возникновение аварий, и разработки мер их предотвращения.
## 2. Цель и Мотивация 
Основная цель проекта - выявить основные причины аварий, время и места их частых возникновений, чтобы помочь снизить колличество аварийно опасных зон в городе и выявить возмоожность предотвращения аварий в зависимости от причины их возникновения.
## 3. Источники Данных 
Данные были получены с Kaggle и содержат информацию о дорожно-транспортных происшествиях в Нью-Йорке за период с 2012 по 2014 гг. Исходный датасет включал ~2 000 000 строк и 29 столбцов, охватывающих такие аспекты, как дата аварии, время аварии, местоположение, количество пострадавших, количество погибших, причина, тип транспорта и т.д.
